Getting Started with Go

Go (Golang) is a powerful programming language that's perfect for building scalable applications. Here's a comprehensive guide to get you started with Go development.

Installation and Setup

  • Install Go from the official website
  • Set up your GOPATH environment variable
  • Verify installation with go version

Creating Your First Project

Follow these steps to create and run your first Go project:

  1. Initialize project: go mod init project_name - Creates a new Go module
  2. Create main file: Create a main.go file in your project directory
  3. Build project: go build - Creates an executable binary file
  4. Run project: go run main.go - Executes your Go program

Live Reload with Air

For development efficiency, use Air for live reloading:

  1. Install Air: go install github.com/air-verse/air@latest
  2. Initialize config: air init - Creates .air.toml configuration file
  3. Run with config: air -c .air.toml - Starts live reload server
  4. Custom port: air server --port 8080 - Run on specific port

Best Practices

When working with Go, remember to follow Go conventions for naming, use go fmt for formatting, and leverage Go's built-in testing framework. The language's simplicity and performance make it ideal for building microservices, CLI tools, and web applications.

Quick Reference

  • go mod init - Initialize module
  • go build - Compile project
  • go run - Run project
  • go test - Run tests
  • go fmt - Format code
